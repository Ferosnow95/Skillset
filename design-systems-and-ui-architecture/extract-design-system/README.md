## What it does
Extracts design tokens, component patterns, naming conventions, and visual foundations from an existing codebase (or any public website — colors, typography, spacing, border radius, shadows) and generates JSON / CSS custom properties.

## When to use it
When inheriting products or auditing mature systems.

## Why it matters
Most teams don't actually know the system they have.

## Curator's take
> One of the fastest ways to understand design debt.
