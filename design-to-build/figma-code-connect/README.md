## What it does
Connects Figma components directly to production code.

## When to use it
During implementation and handoff.

## Why it matters
Reduces divergence between design and engineering.

## Curator's take
> Less flashy than AI generation. Far more useful.

## How to get it
Set up the Figma MCP server ([figma/mcp-server-guide](https://github.com/figma/mcp-server-guide)) and Code Connect ([figma/code-connect](https://github.com/figma/code-connect)) to map components to their code implementations.
