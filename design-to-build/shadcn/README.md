## What it does
A collection of accessible, beautifully designed components built on Radix UI and Tailwind CSS. Unlike a traditional library, you copy components directly into your project — you own the code and can modify anything. Pick a style and download your base system at [ui.shadcn.com/create](https://ui.shadcn.com/create).

## When to use it
When you're focused on testing or prototyping a system solution or infrastructure and you want a clean, ready-made UI without spending time on component design decisions that don't matter yet.

## Why it matters
Prototyping stalls when UI work becomes a distraction. shadcn gives you a complete, accessible component set you can drop in immediately — so the focus stays on the thing you're actually trying to validate.

## Curator's take
> The "copy, don't install" model is underrated. You're not locked into a package's opinions — you get the component, you own it, you modify it. That's the right tradeoff for design-adjacent prototyping.

## How to get it
Use the CLI directly: `npx shadcn@latest init` then `npx shadcn@latest add button`. See [ui.shadcn.com](https://ui.shadcn.com).
